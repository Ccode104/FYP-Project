# Code Submission Quick Start Guide

## For Faculty

### Step 1: Create a Code Assignment
1. Log in as faculty
2. Navigate to your course offering
3. Click on "Manage Assignment" tab
4. Click the "üíª Code-based" button
5. Fill in the form:
   - Assignment title (e.g., "Python Basics - Hello World")
   - Description (optional)
   - Release date (optional, YYYY-MM-DDTHH:mm format)
   - Due date (optional, YYYY-MM-DDTHH:mm format)
   - Max score (default: 100)
   - Allow multiple submissions checkbox
6. Click "Create Code Assignment"

### Step 2: View Student Submissions
1. Click on "Submissions" tab
2. Select your code assignment from the dropdown
3. You'll see a list of submissions with "üíª Code submission" indicator
4. Click "View Code" button to review student code

### Step 3: Grade Code Submissions
1. In the code viewer modal:
   - Review the student's code
   - See student name, email, and submission time
   - Click "Copy Code" if you want to test it locally
2. Click "Grade" button
3. Enter score (0-100)
4. Enter feedback (optional)
5. Click OK to save the grade

---

## For Students

### Step 1: Find Code Assignments
1. Log in as student
2. Navigate to your enrolled course
3. View the "Assignments (Present)" tab
4. Look for assignments marked with "(code)" type

### Step 2: Submit Code
1. Scroll to the "Code Submission" section
2. Select the assignment from the dropdown
3. Click "Open Code Editor" button
4. A modal will open with the code editor

### Step 3: Write and Submit
1. Select your programming language from the dropdown (Python, Java, C++, JavaScript, C)
2. Write your code in the textarea
3. Check the line and character count at the bottom
4. Click "Submit Code" button
5. Wait for success notification
6. Close the modal

## Example: Hello World in Python

### Faculty Creates Assignment
```
Title: Python Hello World
Description: Write a Python program that prints "Hello, World!"
Assignment Type: code
Max Score: 100
```

### Student Submits Code
```python
# Student selects Python from language dropdown
print("Hello, World!")
```

### Faculty Reviews
- Views code in modal
- Sees: "Python" as language, student name, submission time
- Grades: 100/100
- Feedback: "Perfect! Well done."

## Common Issues

**Q: I don't see the code editor opening**
- Make sure you've selected an assignment first
- Check that the assignment type is "code"
- Try refreshing the page

**Q: My code submission failed**
- Check your internet connection
- Verify you've written some code (not empty)
- Check browser console for errors
- Try again after a few seconds

**Q: I can't see student submissions**
- Verify you're viewing the correct assignment
- Check that students have actually submitted
- Ensure you have faculty/TA permissions

**Q: Grading doesn't save**
- Verify you entered a valid score (0-100)
- Check your internet connection
- Ensure you have grading permissions

## Keyboard Shortcuts in Code Editor

- **Tab**: Insert 4 spaces (standard indentation)
- **Ctrl/Cmd + A**: Select all code
- **Ctrl/Cmd + C**: Copy code
- **Ctrl/Cmd + V**: Paste code
